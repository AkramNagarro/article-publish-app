# ArticlePublishApp

This project was generated using [Angular CLI](https://github.com/angular/angular-cli) version 21.1.3.

# Project is divided into two layers

1. Frontend - This project's front end part (article-publish-app) is deployed on vercel free host server

2. Backend - Project's backend part (backend) is deployed on render free host server. Backend is implemented using JSON Server (db.json).

# Improtant urls

Github url (frontend changes)- https://github.com/AkramNagarro/article-publish-app

Comment API - https://backend-article-publish-app.onrender.com/comments
